• 后台功能介绍
通知公告
通知公告模块包括：
  • 发送课程公告
  • 对指定的用户发送提醒（目前只支持单个用户的发送）
学生管理
学生管理模块包括：
  • 添加新的用户账号
  • 编辑指定用户的账号资料（包括密码、权限等）
  • 添加授权码（用于学生在前台管理修改自己账户的找回问题和密码，可以设定授权码的有效时间）
主题维护
主题维护模块包括：
  • 添加新的章（单元）
  • 在已有章的基础上添加新的主题（节）
  • 管理已经创建的章、主题
作业管理
作业管理模块包括：
  • 布置作业（单人形式完成、小组形式完成）
  • 管理已经布置的作业（编辑发布信息、作业deadline）
  • 查看学生已经上交的作业（拟支持在线查看，暂时不支持打包下载，打包目前预计采用ftp模式开放）
资源管理
资源管理模块包括：
  • 为已经创建的主题上传对应的资源（必须要有主题才可以上传）
  • 管理已经上传的资源
关于讨论
讨论目前作为一个独立的板块，相关管理选项集成在前台的讨论板块中，不需要后台管理。
